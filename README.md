# gotranslate
Go version of [DavideGalilei/gpytranslate](https://github.com/DavideGalilei/gpytranslate)

---
## Features
 - **Typed options & results**
 - **Supports emoji**
 - **Free to use**
 - **Easy**

----
## Quick Start

### Installation
Requirements:
- Go 1.10 or newer.


```bash
$ go get -u github.com/roj1512/gotranslate
```
